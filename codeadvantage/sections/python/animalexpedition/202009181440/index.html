Lesson 1 | Sep. 18

Code

  • main.py -- basic turtle functions

Homework

Part 1

Please do this!
  1. Install Python 3 on your computer. This guide may be helpful.
  2. Create a folder in which to store this class's projects.
  3. Load up IDLE, Python's built-in editor (or your preferred code editor), and copy the code from class (linked above) to a .py file in that folder.
  4. Save it and run it to make sure everything is working.

Part 2

Using what we did in class as a jumping-off point, create a simple Turtle program where you can move the turtle around using the arrow keys or WASD.

The function to take key presses as input is screen.onkeypress(func, key).

Arrow keys are strings "Left", "Right", "Up", and "Down".

There are many ways to move the turtle. For example, you can use t.goto(x,y), t.setpos(x,y), t.forward/backward(x,y), and t.left/right(deg).

Try writing your own function to get the turtle to move around in more complex ways, or even draw a shape when you right-click (button 2).

If you get stuck or you don't know how to do something, consult the Turtle library documentation!

Lesson 3 | Oct. 2

Code

  • week3.py -- creating and moving along a grid

Homework

For the following exercises, you can try writing code or just plan it out on paper!

  • Devise a way to spawn the food, for the snake to eat it, and for it to reappear somewhere else on the screen.
  • Devise a way to draw the snake's tail following it along its path.
  • Devise a way to check for collisions with the wall or with your own tail.

It's okay if you get stuck! I just want you to get thinking about how to implement these aspects of the game.

This is more about designing than worrying about exactly how to code it.
