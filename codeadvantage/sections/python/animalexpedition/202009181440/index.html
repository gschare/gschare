    Part 2

    Using what we did in class as a jumping-off point, create a simple Turtle program where you can move the turtle around using the arrow keys or WASD.
    The function to take key presses as input is screen.onkeypress(func, key).
    Arrow keys are strings "Left", "Right", "Up", and "Down".
    There are many ways to move the turtle. For example, you can use t.goto(x,y), t.setpos(x,y), t.forward/backward(x,y), and t.left/right(deg). Try writing your own function to get the turtle to move around in more complex ways, or even draw a shape when you right-click (button 2).
    If you get stuck or you don't know how to do something, consult the Turtle library documentation!
